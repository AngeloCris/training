Condicionales

Nos permite controlar la manera en c√≥mo se ejecuta nuestro c√≥digo (establecen condicionales).

Las estructuras de control m√°s usadas en R son las siguiente:

if, else

if se usa cuando deseamos que una operaci√≥n se ejecute cuando cumple una condici√≥n deseada (Ej. Si esta condici√≥n es cierta, entonces haz una operaci√≥n espec√≠fica)

else se usa para indicarle a R qu√© hace en caso de que la condici√≥n de un if no se cumpla.

La estructura de un la funci√≥n if es la siguiente

#if (condicion) {
#  proceso_si_se_cumple_condicion
#}

Si la condici√≥n es verdadera, entonces se realiza la operaci√≥n. En caso contrario, no se realiza la operaci√≥n. Veamos algunos ejemplos:

#Se cumple la condici√≥n y se muestra "verdadero".
if (19 > 15) {
  "verdadero"
}
## [1] "verdadero"
#Si no se cumple la condici√≥n, el c√≥digo no se ejecuta.
if (10 > 15) {
  "verdadero"
}

la funci√≥n else complementa a if, es decir, le asigna un proceso siempre en cuando la condici√≥n inicial no se cumpla. En otras palabras, un if con else es la manera de decirle a R:

  • SI esta condici√≥n es cierta, ENTONCES haz estas operaciones.DE OTRO MODO haz estas otras operaciones.

La estructura de in if con else es el siguiente:

#if (condition) {
#  proceso1
#} else {
#  proceso2
#}

Del ejemplo anterior podemos hacer las siguientes modificaciones:

#Se cumple condici√≥n
if (18 > 15) {
  "Verdadero"
} else {
  "Falso"
}
## [1] "Verdadero"
#No se cumple condici√≥n
if (10 > 15) {
  "Verdadero"
} else {
  "Falso"
}
## [1] "Falso"

Para ilustrar el uso de if y else definiremos una funci√≥n que calcule el promedio de calificaciones de un estudiante y, dependiendo de la calificaci√≥n calculada, nos devuelva un mensaje espec√≠fico.

promedio <- 
  function(nota){
    mean(nota)
  }

promedio(c(5,8,9,6,5))
## [1] 6.6

Ahora realizaremos un proceso de tal forma que la funci√≥n nos muestre si un estudiante ha aprobado o no. Si asumimos que un estudiante necesita obtener 5 o m√°s en promedio para aprobar podemos decir:

  • SI el promedio de un estudiante es igual o mayor a 5, ENTONCES mostrar ‚ÄúAprobado‚Äù, DE OTRO MODO, mostrar ‚ÄúReprobado‚Äù.
promedio <- 
  function(notas) {
    media <- mean(notas)
    
    if (media >= 5) {
      print("Aprobado")
    } else{
      print("Desaprobado")
    }
  }
promedio(c(6,4,5,7,5,6))
## [1] "Aprobado"
promedio(c(3,4,4,4,5,3))
## [1] "Desaprobado"

If else

La funci√≥n ifelse() permite vectorizar if, else. En lugar de escribir una l√≠nea de c√≥digo para cada comparaci√≥n, podemos usar una sola llamada a esta funci√≥n, que se aplicar√° a todos los elementos de un vector.

Si intentamos usar if else con un vector, se nos mostrar√° una advertencia:

# if (1:10 < 3) {
#   "Verdadero"
# }

El mensaje muestra que la condici√≥n s√≥lo es evaluada para el primer elemento del vector 1:10, es decir para 1. Los dem√°s elementos son ignorados.

Por el contrario, con ifelse le indicamos que la condici√≥n debe de cumplirse para cada uno de los elementos del vector. Esta funci√≥n tiene tres argumentos:

  • un condicional

  • un valor que indique qu√© mostrar si la condici√≥n se cumple

  • un valor que indique qu√© mostrar si la condici√≥n no se cumple

vector <- 1:10
ifelse(vector > 5, "TRUE", "FALSE")

For

La estructura for permite ejecutar bucles realizando una operaci√≥n para cada elemento de un conjunto de datos, es decir, hace un proceso de iteraci√≥n.

#la estructura es
#for (variable in vector) {
#  proceso
#}

Esta estructura le dice a R:

  • PARA cada elemento EN un objeto, realice un proceso espec√≠fico.

Cuando observamos la estructura del bucle, la ‚Äúvariable‚Äù puede ser un elemento cualquiera (la podemos llamar como nosotros queramos) pero el objeto ‚Äúvector‚Äù tiene que ser un objeto existente.
