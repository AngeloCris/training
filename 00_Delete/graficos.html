FUNCI√ìN PLOT

La funci√≥n plot() es usada de manera general para crear gr√°ficos en R. As√≠ mismo, podemos ajustar diferentes par√°metros para controlar el aspecto y dimensi√≥n de los gr√°ficos.

plot() siempre pide un argumento x, que corresponde al eje X de una gr√°fica. x requiere un vector y si no especificamos este argumento, obtendremos un error y no se crear√° la gr√°fica. El resto de los argumentos de plot() son opcionales, pero el m√°s importante es y. Este argumento tambi√©n requiere un vector y corresponde al eje Y de la gr√°fica.

Dependiendo del tipo de dato que sea x o y tendremos un gr√°fico espec√≠fico de acuerdo a las siguientes reglas:

En donde los tipos de dato son:

  • Continuo: Un vector num√©rico, entero, l√≥gico o complejo.

  • Discreto: Un vector de factores o cadenas de texto.

Hay funciones que generan tipos espec√≠ficos de gr√°ficos como por ejemplo la funci√≥n barplot() o hist().

Adem√°s, es importante tener en cuenta que existen funciones de ALTO NIVEL y funciones de BAJO NIVEL. Las primeras permiten crear un nuevo gr√°fico, las segundas, agregar elementos a uno preexistente.

Histogramas

Permite observar la distribuci√≥n de los datos num√©ricos usando barras. Un histograma se crea con la funci√≥n hist(), donde el primer argumento debe ser un vector num√©rico.

Un ejemplo con la base de datos anteriormente usada:

# histograma
hist((x = banco$age))

# modificar algunos par√°metros
hist(x = banco$age, 
     # titulo
     main = "Histograma de edad",
     # ejes
     xlab = "Edad", 
     ylab = "Frecuencia",
     # color
     col = "purple")

Se pude obtener una gran variedad de colores en R, pueden ver una lista de los colores disponibles en R en el siguiente enlace: http://www.stat.columbia.edu/~tzheng/files/Rcolor.pdf

Diagrama de dispersi√≥n

Usado para mostrar la relaci√≥n entre dos variables num√©ricas continuas usando puntos. En R para generar un diagrama de dispersi√≥n se usa la funci√≥n plot() con sus respectivos argumentos x e y.

Tenemos valores extremos que no permiten visualizar correctamente la mayor cantidad de nuestros datos. Entonces, para fines de tener una gr√°fica m√°s informativa, vamos a cambiar todos los valores mayores a 15000.

estos valores para tener una mejor visualizaci√≥n.

plot(x = banco$age, 
     y = banco$balance, 
     type = "p",
     pch = "+")

plot(x = banco$age, y = banco$balance, col = as.factor(banco$loan))

Sabemos que los niveles de loan son ‚Äúno‚Äù y ‚Äúyes‚Äù, adem√°s que los colores han sido rojo y negro pero ser√≠a de utilidad para el lector agregar una leyenda que explique expl√≠citamente los colores.

Diagrama de cajas

Son gr√°ficos que muestran la distribuci√≥n de una variable usando cuartiles, de modo que de forma visual podemos inferir algunas cosas sobre su dispersi√≥n, ubicaci√≥n y simetr√≠a.

Para crear un boxplot usamos la funci√≥n plot() teniendo en cuenta que el argumento x debe ser un factor, y como argumento y, un vector num√©rico.

plot(x = as.factor(banco$education),
     y = banco$age)

plot(x = as.factor(banco$education),
     y = banco$age,
     main = "Edad por nivel educativo",
     xlab = "Nivel educativo", 
     ylab = "Edad",
     col = c("orange3",
             "yellow3",
             "green3",
             "grey"))

Tambi√©n se puede generar el mismo gr√°fico con una funci√≥n espec√≠fica de R denominada boxplot()

Exportar gr√°ficos

Para exportar un gr√°fico se puede usar algunas de las siguientes funciones:

  • bpm()

  • jpeg()

  • pdf()

  • png()

  • tiff()

Cada una de estas funciones tiene los siguientes argumentos:

  • filename: Nombre y ruta del plot.

  • width: El ancho de la imagen.

  • height: El alto de la imagen.

Estas funciones se llaman antes de crear el gr√°fico. Veamos un ejemplo:
